# Hey, I'm Mario 👋

Senior Software Engineer and AI specialist who builds at the intersection of intelligent systems and real products. Founder of **Syntonixx LLC** — an AI integration and software consulting firm based in Arizona. MS in AI from Arizona State University.

## 🧠 What I Build

- **OrchestratAI** — Multi-agent AI platform with RAG/CAG routing, ChromaDB vector search, Redis semantic caching, and real-time streaming responses
- **PrismIQ** — AI-powered dynamic pricing engine with customer segmentation using K-Means, XGBoost, LangChain, and SHAP explainability (hackathon winner)
- **CRM Platform Plugins** — HubSpot & GoHighLevel marketplace integrations with webhook-driven data sync and multi-provider API orchestration
- **Operational Dashboards** — Real-time data visualization platforms with SQL-backed reporting, SSE/WebSocket feeds, and windowed rendering for high-throughput data

## 🔬 AI & Engineering Highlights

- Built agentic workflows with LangGraph and LangChain across multiple production systems

- Scaled real-time systems to 200k+ concurrent users with 99.9% uptime
